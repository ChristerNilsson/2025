# Berger

Skriv in namnen i en textfil

```
&p1=1698 Onni Aikio
&p2=1558 Helge Bergström
&p3=1549 Jonas Hök
```

Markera allt i textfilen med ctrl-a

Gå till Chrome och klistra in med ctrl-v

Skriv ut sidan med ctrl-p

Skriv in de vita resultaten i filen

```
&r1=10rx1
&r2=0r101
```

Förklaring
```
r1 = rond 1

1 = vit vinst
0 = vit förlust
r = remi
x = partiet ej spelat
```
